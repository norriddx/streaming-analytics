# streaming-analytics
end-to-end streaming platform analysis: advanced SQL (cohorts, retention, RFM, funnels) + Tableau dashboard

**tools:** PostgreSQL · Python (pandas, seaborn) · Tableau Public

## Data
 
This project uses the [KKBox Churn Prediction Challenge](https://www.kaggle.com/c/kkbox-churn-prediction-challenge/data) dataset from Kaggle.
 
### Required files
 
- `members_v3.csv`
- `transactions_v2.csv`
- `user_logs_v2.csv`
 
### Setup
 
1. Download the files from the Kaggle competition page (requires a Kaggle account).
2. Place them in the `data/` folder at the project root.
3. The `data/` directory is listed in `.gitignore` and is not tracked by Git.
